# Code of Conduct

## Our Pledge

We are committed to maintaining a professional, respectful, and safe environment for all contributors and participants.

## Our Standards

**Expected behaviour:**
- Professional and respectful communication.
- Constructive criticism focused on ideas, not people.
- Accepting responsibility for mistakes and learning from them.
- Prioritising security and quality of the product.

**Unacceptable behaviour:**
- Harassment, discrimination, or personal attacks.
- Sharing confidential project information outside authorised channels.
- Bypassing security controls or code review processes.
- Committing untested or deliberately obfuscated code.

## Enforcement

Violations should be reported to the project lead. Confirmed violations may result in removal from the project.

## Scope

This Code of Conduct applies to all project spaces: code reviews, issue trackers, internal chats, and in-person interactions related to the project.
